# Gita Kids — Learn with Krishna & Madhwacharya

Interactive prototype of an iOS learning app for kids: Bhagavad Gita shlokas with karaoke chanting,
the life of Sri Madhwacharya, Dvaita (Tattvavada) philosophy, Mahabharata Tatparya Nirnaya secrets,
Sri Raghavendra Swamy's teachings, quizzes, points and badges.

**Hari Sarvottama · Vayu Jeevottama**

## Run
Open `index.html` in any browser — fully self-contained, no dependencies.
